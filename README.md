# Rewind

Backtest historical trading strategies

## Install

Add `rewind` to your list of dependencies in `mix.exs`

```elixir
def deps do
  [
    {:rewind, "~> 0.0.1"}
  ]
end
```

## Requirements

`rewind` requires Elixir 1.12+ & Erlang/OTP 22+
